## 1. RESUMEN DEL PROYECTO

ALPO es una herramienta de protección digital para detectar señales tempranas
de reclutamiento criminal en interacciones de menores. El MVP:
  1. Recibe un audio grabado/subido desde el navegador.
  2. Lo transcribe a texto (Whisper API vía OpenAI o HuggingFace).
  3. Busca la frase de riesgo: "dinero fácil" (case-insensitive, sin tilde también).
  4. Si hay match: genera un hash SHA-256 del texto + timestamp, almacena ese
     hash en la blockchain de Monad Testnet mediante un smart contract, devuelve
     el txHash, y muestra todo en la UI con una alerta visible.
  5. Registra el evento en PostgreSQL para trazabilidad interna.

NO se requiere login, no se requiere wallet del usuario final: el backend firma
la transacción con una wallet propia del sistema.

## 2. FLUJO COMPLETO (Happy Path)

  [Usuario] → graba o sube audio (.webm/.mp4/.mp3/.wav)
      ↓
  [Frontend React] → POST /api/analyze  (multipart/form-data, campo: "audio")
      ↓
  [Backend Node/Express]
    a) Recibe audio → llama Whisper API → obtiene transcripción (string)
    b) Normaliza texto (lowercase, quita tildes opcionales)
    c) Busca "dinero facil" / "dinero fácil" en el texto
    d) Si NO hay match → responde { detected: false, transcription }
    e) Si SÍ hay match:
       - Construye payload: { transcription, timestamp: Date.now(), keyword: "dinero fácil" }
       - Calcula hash SHA-256 del JSON.stringify(payload)
       - Llama al smart contract: storeEvidence(bytes32 hash, uint256 timestamp)
       - Guarda en PostgreSQL: tabla `evidence_logs` (ver esquema abajo)
       - Responde { detected: true, transcription, hash, txHash, timestamp }
      ↓
  [Frontend] → muestra resultado en Pantalla 2:
    - Transcripción completa
    - Badge rojo "⚠ FRASE DETECTADA: dinero fácil"
    - Hash generado
    - TxHash con link al explorador de Monad Testnet
    - Alerta: "Tutor notificado" (simulado en UI, sin email real en demo)

## 4. SMART CONTRACT

### 4a. AlpoEvidence.sol

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.28;

contract AlpoEvidence {
    event EvidenceStored(
        bytes32 indexed evidenceHash,
        uint256 timestamp,
        address indexed sender
    );

    struct EvidenceRecord {
        bytes32 evidenceHash;
        uint256 timestamp;
        address sender;
    }

    EvidenceRecord[] public records;

    function storeEvidence(bytes32 evidenceHash, uint256 timestamp) external {
        records.push(EvidenceRecord({
            evidenceHash: evidenceHash,
            timestamp: timestamp,
            sender: msg.sender
        }));
        emit EvidenceStored(evidenceHash, timestamp, msg.sender);
    }

    function getRecord(uint256 index) external view returns (EvidenceRecord memory) {
        return records[index];
    }

    function totalRecords() external view returns (uint256) {
        return records.length;
    }
}
```

### 5e. services/detector.js

```js
const KEYWORDS = ["dinero fácil", "dinero facil"];

export function detectKeyword(text) {
  const normalized = text.toLowerCase()
    .normalize("NFD")
    .replace(/[\u0300-\u036f]/g, ""); // quita tildes
  const match = KEYWORDS.find(kw => {
    const kwNorm = kw.normalize("NFD").replace(/[\u0300-\u036f]/g, "");
    return normalized.includes(kwNorm);
  });
  return match || null;
}
```

### 5f. services/hasher.js

```js
import crypto from "crypto";

export function generateHash(payload) {
  // payload = { transcription, timestamp, keyword }
  return crypto
    .createHash("sha256")
    .update(JSON.stringify(payload))
    .digest("hex");
}
```

### 5g. services/blockchain.js

```js
import { createWalletClient, createPublicClient, http, parseAbiItem } from "viem";
import { privateKeyToAccount } from "viem/accounts";
import { monadTestnet } from "viem/chains";

const CONTRACT_ABI = [
  {
    name: "storeEvidence",
    type: "function",
    stateMutability: "nonpayable",
    inputs: [
      { name: "evidenceHash", type: "bytes32" },
      { name: "timestamp", type: "uint256" },
    ],
    outputs: [],
  },
];

export async function storeEvidenceOnChain(hexHash, timestamp) {
  const account = privateKeyToAccount(process.env.PRIVATE_KEY);

  const walletClient = createWalletClient({
    account,
    chain: monadTestnet,
    transport: http(process.env.MONAD_RPC_URL),
  });

  // hexHash viene como string hex de 64 chars (SHA-256), convertir a bytes32
  const bytes32Hash = `0x${hexHash}`;

  const txHash = await walletClient.writeContract({
    address: process.env.CONTRACT_ADDRESS,
    abi: CONTRACT_ABI,
    functionName: "storeEvidence",
    args: [bytes32Hash, BigInt(timestamp)],
  });

  return txHash;
}
```

### 5h. routes/analyze.js (lógica principal)

```js
import express from "express";
import multer from "multer";
import path from "path";
import fs from "fs";
import { transcribeAudio } from "../services/transcription.js";
import { detectKeyword } from "../services/detector.js";
import { generateHash } from "../services/hasher.js";
import { storeEvidenceOnChain } from "../services/blockchain.js";
import { pool } from "../db/client.js";

const router = express.Router();
const upload = multer({ dest: "/tmp/alpo-uploads/" });

router.post("/analyze", upload.single("audio"), async (req, res) => {
  const filePath = req.file?.path;
  if (!filePath) return res.status(400).json({ error: "No audio file received" });

  try {
    // 1. Transcribir
    const transcription = await transcribeAudio(filePath);

    // 2. Detectar
    const keyword = detectKeyword(transcription);

    if (!keyword) {
      return res.json({ detected: false, transcription });
    }

    // 3. Hash
    const timestamp = Date.now();
    const payload = { transcription, timestamp, keyword };
    const hash = generateHash(payload);

    // 4. Blockchain
    const txHash = await storeEvidenceOnChain(hash, timestamp);

    // 5. PostgreSQL
    await pool.query(
      `INSERT INTO evidence_logs
       (transcription, keyword_detected, evidence_hash, tx_hash, timestamp)
       VALUES ($1, $2, $3, $4, $5)`,
      [transcription, keyword, hash, txHash, timestamp]
    );

    return res.json({ detected: true, transcription, keyword, hash, txHash, timestamp });

  } catch (err) {
    console.error(err);
    return res.status(500).json({ error: err.message });
  } finally {
    // Limpiar archivo temporal
    if (filePath) fs.unlinkSync(filePath);
  }
});

export default router;
```

### 6b. AudioRecorder.jsx (skeleton)

```jsx
import { useState, useRef } from "react";

export function AudioRecorder({ onAudioReady }) {
  const [recording, setRecording] = useState(false);
  const mediaRecorderRef = useRef(null);
  const chunksRef = useRef([]);

  const startRecording = async () => {
    const stream = await navigator.mediaDevices.getUserMedia({ audio: true });
    const mediaRecorder = new MediaRecorder(stream);
    mediaRecorderRef.current = mediaRecorder;
    chunksRef.current = [];

    mediaRecorder.ondataavailable = (e) => chunksRef.current.push(e.data);
    mediaRecorder.onstop = () => {
      const blob = new Blob(chunksRef.current, { type: "audio/webm" });
      onAudioReady(blob);
    };

    mediaRecorder.start();
    setRecording(true);
  };

  const stopRecording = () => {
    mediaRecorderRef.current?.stop();
    setRecording(false);
  };

  return (
    <button onClick={recording ? stopRecording : startRecording}>
      {recording ? "⏹ Detener grabación" : "🎙 Grabar Audio"}
    </button>
  );
}
```

### 6c. App.jsx (flujo principal skeleton)

```jsx
import { useState } from "react";
import { AudioRecorder } from "./components/AudioRecorder";

const API_URL = import.meta.env.VITE_API_URL || "http://localhost:3001";

export default function App() {
  const [audioBlob, setAudioBlob] = useState(null);
  const [loading, setLoading] = useState(false);
  const [result, setResult] = useState(null);

  const handleAnalyze = async () => {
    if (!audioBlob) return;
    setLoading(true);
    const formData = new FormData();
    formData.append("audio", audioBlob, "recording.webm");

    const res = await fetch(`${API_URL}/api/analyze`, {
      method: "POST",
      body: formData,
    });
    const data = await res.json();
    setResult(data);
    setLoading(false);
  };

  return (
    <div>
      <h1>ALPO – Protección Digital</h1>
      <AudioRecorder onAudioReady={setAudioBlob} />
      {/* FileUploader también aquí */}
      <button onClick={handleAnalyze} disabled={!audioBlob || loading}>
        {loading ? "Analizando..." : "Analizar"}
      </button>
      {result && <ResultPanel result={result} />}
    </div>
  );
}
```

## 11. NOTAS PARA EL MODELO QUE IMPLEMENTE ESTO

- NO uses `forge create`; usa siempre `forge script` para deployar.
- NO definas cadena Monad personalizada en el frontend; importa `monadTestnet`
  directamente desde `viem/chains`.
- NO uses `--no-commit` en `forge init`; ese flag no existe en versiones recientes.
- La wallet del sistema (que firma las tx) es una wallet controlada por el backend.
  El usuario final NO necesita MetaMask ni wallet propia en el demo.
- El hash que va al contrato es bytes32. SHA-256 produce 32 bytes → encaja perfecto.
  Pásalo como `0x` + los 64 caracteres hex del hash.
- Si Whisper API no está disponible, usar HuggingFace Inference API con el modelo
  openai/whisper-large-v3 (ver sección 5d, alternativa sin OpenAI).
- Todos los errores del backend deben retornar JSON { error: string }, nunca HTML.
- El frontend debe mostrar errores de red de forma amigable, no stack traces.
- Para el demo es suficiente con "dinero fácil" como única keyword. La arquitectura
  (array KEYWORDS en detector.js) ya está preparada para escalar a más frases.
